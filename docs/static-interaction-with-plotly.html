### Relationship between penguin bill length and depth
  
```{r}
ggplot(penguins, aes(x = bill_length_mm, y = bill_depth_mm)) +
  geom_point(size = 2, color = "#359EAAF") +
  theme_minimal() +
  labs(x = "Length (mm)", y = "Depth (mm)") 
```

We could of course make it interactive in ways that rely on HTML, like plotly.

```{r}
ggplotly(ggplot(penguins, aes(x = bill_length_mm, y = bill_depth_mm)) +
  geom_point(size = 2, color = "#359EAAF") +
  theme_minimal() +
  labs(x = "Length (mm)", y = "Depth (mm)"))
```

And play around with the tooltip argument in ggplotly to adjust what users see

```{r}
ggplotly(ggplot(penguins, aes(x = bill_length_mm, y = bill_depth_mm)) +
  geom_point(size = 2, color = "#359EAAF") +
  theme_minimal() +
  labs(x = "Length (mm)", y = "Depth (mm)"),
  tooltip = c("x"))
```
